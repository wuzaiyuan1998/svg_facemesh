## 产品介绍

**六角恐龙 FaceMesh 表情捕捉** 是一款基于 Google MediaPipe 技术的实时面部追踪应用。通过摄像头捕捉你的面部表情，实时驱动一只可爱的六角恐龙 SVG 角色，让它模仿你的表情动作。

### 核心特性

- ✅ **实时追踪** - 468 个面部关键点，60+ FPS
- ✅ **表情映射** - 眨眼、张嘴、眼睛追踪、腮红
- ✅ **离线运行** - 无需联网，保护隐私
- ✅ **可爱角色** - 原创六角恐龙 SVG 设计
- ✅ **纯前端** - 无需后端服务器，浏览器即可运行

## 快速开始

### 方式一：直接打开（推荐）

1. 找到 `liujiao-facemesh.html` 文件
2. **双击打开** 或 **右键 → 打开方式 → Chrome/Edge**
3. 点击 **"启动"** 按钮
4. **允许摄像头权限**
5. 面对摄像头，开始互动！

### 方式二：本地服务器（更稳定）

```bash
# 1. 打开终端/命令提示符
# 2. 进入文件所在目录
cd /path/to/kalidokit-svg-demo

# 3. 启动本地服务器（任选一种）

# Python 3
python3 -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080

# Node.js (需要先安装 http-server)
npx http-server . -p 8080

# 4. 浏览器访问
http://localhost:8080/liujiao-facemesh.html
```

### 权限说明

首次运行时，浏览器会请求以下权限：


## 表情映射表

### 1. 👁️ 眼睛追踪

| 你的动作 | 六角恐龙反应 | 技术细节 |
|----------|--------------|----------|
| **左右看** 👀 | 瞳孔跟随移动 | 虹膜关键点 (468, 473) 映射 |
| **眨眼** 👁️ | 眼睛变扁成一条线 | 眼睛开合度 < 0.22 触发 |
| **睁大眼睛** 😲 | 瞳孔恢复正常圆形 | 眼睛开合度 > 0.22 |

**眨眼技巧**：
- 正常眨眼即可触发
- 不需要用力闭眼
- 双眼同时眨眼效果最佳

### 2. 👄 嘴巴表情

| 张嘴程度 | 六角恐龙反应 | 阈值 |
|----------|--------------|------|
| **闭嘴/微笑** 😊 | 红色粗线条弧线 | < 0.025 |
| **微张** 😐 | 浅红色填充下半圆 | 0.025 - 0.04 |
| **大张嘴** 📢 | 深红色填充下半圆（圆润） | > 0.04 |

**模式切换**：
- **线条模式**（闭嘴）：`fill="none"`, `stroke-width="10"`
- **填充模式**（张嘴）：`fill="#E76F51"`, 圆润端点

### 3. 😊 腮红

| 表情 | 六角恐龙反应 |
|------|--------------|
| **微笑** | 腮红变明显（透明度增加） |
| **中性** | 腮红默认透明度 |

### 4. 🌊 外鳃摆动

| 动作 | 六角恐龙反应 |
|------|--------------|
| **头部左右移动** | 左右外鳃相反方向摆动 |
| **静止** | 外鳃保持默认位置 |

## 常见问题

### Q1: 点击"启动"后没有反应？

**A:** 可能是以下原因：

1. **摄像头权限未授予**
   - 检查浏览器地址栏是否有摄像头图标
   - 点击图标，选择"始终允许"

2. **摄像头被其他程序占用**
   - 关闭 Zoom、腾讯会议等使用摄像头的程序
   - 刷新页面重试

3. **浏览器不支持**
   - 使用 Chrome/Edge 最新版
   - 不要使用 IE 浏览器

### Q2: 检测不到面部？

**A:** 请尝试以下方法：

1. **调整光线**
   - 确保面部光线充足
   - 避免背光（不要背对窗户）

2. **调整距离**
   - 距离摄像头 30-50cm
   - 面部占画面 1/3 左右

3. **正面朝向**
   - 不要侧脸或低头
   - 保持面部完整在画面中

4. **摘掉遮挡物**
   - 摘掉口罩、墨镜
   - 刘海不要遮挡眼睛

### Q3: 表情不灵敏？

**A:** 调整建议：

1. **眨眼不触发**
   - 稍微用力闭眼
   - 确保双眼都在画面中

2. **嘴巴不变色**
   - 张大一些（阈值 0.025）
   - 确保摄像头能清晰看到嘴部

3. **瞳孔不移动**
   - 头部保持稳定
   - 只转动眼球

### Q4: FPS 很低/卡顿？

**A:** 优化建议：

1. **关闭其他标签页** - 释放内存
2. **降低摄像头分辨率** - 代码中修改 `width: 640, height: 480`
3. **使用独立显卡** - 在笔记本上切换显卡
4. **重启浏览器** - 清理缓存

### Q5: 离线版本无法加载？

**A:** 检查以下文件是否完整：

```
liujiao-facemesh-offline/
├── liujiao-facemesh.html
├── face_mesh.js
├── face_mesh_solution_packed_assets.data
└── face_mesh_solution_simd_wasm_bin.wasm
```

确保所有文件在同一目录下。
